# unraid-setup-guides
Simple to follow setup guides for Unraid and apps

## Use of this server

Personal file access from LAN and WAN via Mounted Network Drives and Tailscale

Jellyfin Server - Movies and TV Shows

Immich Server - Photos Backup

PiHole - Adblocked and Telemetry Blocker

## My Hardware

- Dell Options Micro or SFF (Intel i5 gen 8 and above for iGPU) 
- 16GB RAM
- 1TB SSD - Files only
- 256GB SSD - Docker, VMs
